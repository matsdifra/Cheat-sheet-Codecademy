Form Tags

Tag Name Description
<input> Input The input element is used to render a variety of input fields on a webpage, including text fields, checkboxes, buttons, etc.
<label> Label The label element provides identification for a specific <input>, based on matching values of the input's id attribute and the label's for attribute.
<form> Form The HTML <form> element is used to collect and send information to an external source
<select> Select This is used to create a dropdown list
<option> Option One or more <option> Can be nested within <select> to create a list of choices for the dropdown list
<datalist> Datalist This can be used to implement a search/autocomplete functionality. An <input> must be paired with a <datalist>. The <input>'s list value must match the value of the id of the <datalist>. The datalist element can store a list of <option>s.
<textarea> Textarea The textarea element is used when creating a text-box for multi-line input (e.g. a comment section)
